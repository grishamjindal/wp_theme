-------------------------------------------------------
    Installation
-------------------------------------------------------

1. Upload the folder to the "/wp-content/themes/" directory
2. Activate the Theme through the 'Themes' menu in WordPress
3. Go to "Appearance" > Theme options
4. Config theme as you need.
